# Cruise Admin Dashboard

## Overview
The Cruise Admin Dashboard is an Angular 20 application built to practice modern frontend development concepts including:
- CRUD Operations
- State management
- Reactive programming
- API intergration

This allows users to:
- View customer cruise enquiries
- Add new enquiries
- Update enquiry statuses
- Delete enquiries
- View travel updates from an external API

## Technologies
- Angular 20
- TypeScript
- RxJS
- HTML5
- CSS3
- Angular Router
- Angular HttpClient

## Features

### Enquiry Management
- Create new enquiries
- Edit enquiry statuses
- Delete enquiries
- Shared state management

### Dashboard
- API driven travel updates
- Dynamic card rendering
- Observable data flow

### Angular Concepts Practised
- Components
- Services
- Dependency Injection
- Reactive State Management
- BehaviourSubject
- Observables
- HttpClient
- Interfaces
- CRUD Operations

## Installation
- Clone the repository using git clone https://github.com/your-name/cruise-dashboard.git
- Install the dependencies using npm install
- Run the application using ng serve
- go to http://localhost:4200 using your browser
- Username: "admin" | Password: "password123"

## Things learned during this project
- State management using RxJS BehaviourSubject
- Observable subscription patterns
- Angular dependancy injection
- API intergration using HttpClient
- Typescript interfaces and type safety
- CRUD operations using immutable array updates
- Component and service communication
